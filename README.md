A script to write names on image templates.

### Setup environment
```bash
python -m venv venv;
source venv/bin/activate;
```
for windows
```powershell
venv/Scrpits/activate
```

### Run
```bash
cd <project folder>
python main.py <English name> <Arabic name>
```
